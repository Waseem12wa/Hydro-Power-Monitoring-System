🌊 Quick Start Guide

Learn how to use your Hydro Power Monitoring System

1 Dashboard Overview

The dashboard has 4 main tabs:

  • 📊 Overview - Real-time gauges and current readings
  • 📈 History & Graphs - Historical data charts and trends
  • 🤖 AI Insights - AI-powered predictions and analysis
  • ⚠️ Alerts - System warnings and notifications

2 Configuration Page

Click ⚙️ Configuration to set up your plant:

  1. Click 🚀 Quick Create Moolia Plant for instant setup
  2. OR use templates: Basic Electrical, Hydro Power Full, Moolia Plant
  3. OR add custom fields: Click "➕ Add Field" and enter sensor names
  4. Click Save & Start to begin simulation

3 Simulation Control

From the Configuration page:

  • Update Interval: 1-60 seconds (how fast data updates)
  • Variation: 0-20% (random fluctuation to simulate sensors)
  • Start/Stop: Control data generation with one click
  • Status Badge: Shows green (running) or red (stopped)

4 Adding Custom Sensors

You can add ANY sensor type:

  1. Go to Configuration → Configuration Editor
  2. Click ➕ Add Field
  3. Enter Field Name (voltage, pressure, temperature, vibration, etc.)
  4. Enter Value (base value for simulation)
  5. Select Type (Number or Text)
  6. Repeat for all sensors you need
  7. Click Save & Start

5 Understanding AI Insights

The AI tab provides three types of analysis:

  • Efficiency Analysis: Overall plant performance review
  • Anomaly Detection: Real-time deviation alerts
  • Performance Predictions: 24-hour forecasts
💡 Tip: AI models analyze your data in real-time and provide optimization recommendations!

6 Monitored Parameters

Standard Parameters:

  • Voltage (V)
  • Current (A)
  • Power Output (kW)
  • Frequency (Hz)
  • Turbine RPM
  • Water Flow Rate (m³/s)
  • Water Head (meters/feet)
  • Generator Temperature (°C)
  • Efficiency (%)

Plus any custom sensors you add!

7 Managing Configurations

Create multiple configurations for different scenarios:

  • Edit: Click blue pencil icon
  • Delete: Click red trash icon
  • Activate: Click green check icon
  • Switch: Select from dropdown and restart

1. Go to Configuration
2. Click "Quick Create Moolia Plant"
3. Wait 5 seconds
4. Go to Dashboard → See live data!
