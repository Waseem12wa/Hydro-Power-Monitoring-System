Getting Started

🚀 Quick Start Guide

  1. No configurations yet? Scroll down to see "Quick Create Moolia Plant" button
  2. Click it and the system will automatically create and start your plant simulation
  3. OR scroll to the Configuration Editor at the bottom to create a custom configuration
  4. Add your own sensor fields by clicking "➕ Add Field" - it's completely flexible!
  5. Once saved, select it from the dropdown above and click "▶️ Start Simulation"
  6. View live data on the Dashboard → Overview tab
💡 Pro Tip: The "Moolia Plant" template includes all your specifications: 410V, 166A, 0.83 PF, 3-phase, 775 RPM, and more!
